## SunGuard UV Protection App

SunGuard is a web application that helps users monitor UV radiation levels and receive personalised sunscreen recommendations based on real-time UV data.

The system integrates:
- OpenWeather API (UV index)
- Node.js backend recommendation engine
- React frontend dashboard

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
